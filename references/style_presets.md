# 视觉风格预设

三套**成人向**风格，通过 `build_page.py --style` 切换。共同底线：**绝不卡通、绝不幼态**——
水彩故事书 + 动画 IP 那套会让成人觉得幼稚、丢面子，是反向劝退。一个场景固定一种风格，保持一致。

| 预设 | 气质 | 适合 | 关键词 |
|------|------|------|--------|
| `editorial`（默认） | 赤陶 + 米白，Fraunces 衬线标题，杂志编辑感 | 职场、商务、偏正式场景 | 温暖、克制、有质感 |
| `minimal` | 黑白 + 黄色高亮，Hanken 无衬线粗体 | 知识卡、对照表、想要利落锐利 | 高对比、信息密度、KK 表那种爽 |
| `journal` | 暖绿 + 奶油，衬线标题，旅行手帐感 | 旅行、生活、轻松场景 | 亲切、手帐、不端着 |

## 选风格的建议

- 拿不准 → 用 `editorial`，最百搭、最"高级感"。
- 内容偏"纠错对照 / 信息密集" → `minimal`，呼应学习者熟悉的小红书干货表样式。
- 场景是旅行/点单/闲聊等生活向 → `journal`，气质更松弛。

## 关于插图风格（仅当 Phase 4 开启插图时）

插图只用于对话分格，且只在"文字少"的前提下由 AI 生成（见 SKILL Phase 4 的判断）。
插图风格也必须**成人向**，与卡通儿童海报彻底切割。在 `illustration` 的生图 prompt 里统一加上：

- **风格锚点**：`clean modern editorial line illustration, minimal flat vector, muted sophisticated palette`
  （干净的现代编辑插画 / 极简扁平矢量 / 低饱和高级配色）
- **人物**：`adult professionals, neutral realistic proportions`——成年人、正常比例，
  **不要** 大头拟人动物、不要 Q 版、不要儿童绘本质感。
- **强制**：`no text in image`——文字一律交给 HTML 排版，图里不放字（生图模型的字必糊）。
- **比例**：对话分格插图建议 `4:3` 或 `1:1`，配合 build_page 的卡片宽度。

> 记住整个 skill 的判断：**文字密度是分水岭。** 高密度的词/句/对话 → HTML 排版（本 skill 默认）；
> 只有"画面氛围"这种文字少的部分 → 才交给 AI 生图。所以插图是锦上添花，不是主体；
> 没有 API key 时，对话以纯文字呈现，这页依然完整可用。
